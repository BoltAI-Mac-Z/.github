## 💡 Overview

<a href="#boltAI">BoltAI</a> is a native Mac application that brings AI language model access into the macOS environment — connecting to multiple AI providers through a unified native interface with quick-access activation from any application, custom AI personas, and conversation management designed for Mac workflows.

The <a href="#boltAI">multi-provider AI support</a> connects to OpenAI GPT models, Anthropic Claude, Google Gemini, Perplexity, Ollama local models, and other AI providers through a single BoltAI interface — switching between different AI models without managing multiple browser tabs or separate applications for each provider. Each provider connects through its own API key, and the model selection is available per-conversation so the most appropriate model for each task is accessible without reconfiguration. The <a href="#boltAI">quick-access panel</a> activates BoltAI from any Mac application through a configurable keyboard shortcut — a floating panel appears over the current application context for immediate AI interaction without switching away from the active workflow. The panel dismisses with the same shortcut or an escape key, returning focus to the original application without disrupting the working context.

The <a href="#boltAI">custom AI personas and system prompts</a> create specialized AI assistants for different tasks — a code review assistant with specific instructions for the codebase's conventions, a writing assistant tuned to a specific style guide, a technical documentation assistant with domain knowledge context. Each persona provides a pre-configured AI interaction context that produces more focused and relevant responses than a generic chatbot interface. The <a href="#boltAI">inline text assistance</a> processes selected text from any Mac application through AI without copying and pasting into a separate chat window — selecting text in any document or interface, triggering BoltAI, and applying AI transformations to the selected content directly.

<a href="#boltAI">Conversation history and management</a> saves AI conversations organized by date and topic — searchable conversation history that makes previous AI interactions retrievable rather than lost when the chat session closes. <a href="#boltAI">Context-aware assistance</a> passes the current application context and selected text to the AI automatically, enabling relevant assistance without manually describing what the user is working on.

<a href="#boltAI">Ollama local model support</a> connects to locally running AI models through the Ollama runtime — AI assistance without sending data to cloud services for privacy-sensitive use cases where local model quality is sufficient. <a href="#boltAI">Setapp subscription availability</a> includes BoltAI in the Setapp toolkit for subscribers who want AI assistance alongside other Mac productivity tools through a single subscription.
